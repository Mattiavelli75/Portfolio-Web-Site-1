My name is Matthew Williams and the purpose of this web-site is to showcase my new software development skills gained from participating in the Just-IT Software Development Boot-Camp.

I hope you enjoy your stay.

Projects:

MySQLDB: Student Enrolment System

I developed a MySQL and PL/SQL student enrollment program that leverages DDL for creating and managing database structures, including tables and relationships. Using DML, the system handles student registrations, course assignments, and updates efficiently.

Javascript: Synthesizer Application

I created a JavaScript synthesizer app with a full octave keyboard, waveform selector (sine, square, sawtooth, triangle), unison effects for layered sound, and a delay effect. It's user-friendly for both beginners and pros to explore music and sound design.

Python: Weather Forecast Application

My Python weather forecast app uses a dicts.json file to handle weather data for almost 50,000 cities globally. It allows users to add, remove, view single entries, or browse all forecasts for international cities, making it a versatile tool for tracking weather conditions worldwide.
